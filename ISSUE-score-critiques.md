## TL;DR
Extract and display 1-5 scores from critiques to show which original response (Claude vs GPT) is rated higher.

## Current State
Critiques contain text feedback but no visible scores. Users can't quickly see which response scored better.

## Desired State
Critiques include 1-5 scores for the responses being critiqued. Scores are extracted and displayed prominently so users can see which response is more credible.
